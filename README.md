# C23 dat2csv

This script parses the numerical results from *DAT* files generated by the **Properties** program of **CRYSTAL23** Quantum Package and generates separed *CSV* files (for spin-polarized solutions).

The structure of the CSV files generated by the script includes headers, making the files ready for import in OriginLab 2024.

## Supported result files

| Calculation type | Default file name |
| :--- | :--- |
| Electronic Band Structure | BAND.DAT |
| Density of States | DOSS.DAT |
| Crystal Orbital Hamiltonian Population | COHP.DAT |
| Crystal Orbital Overlap Population | COOP.DAT |
