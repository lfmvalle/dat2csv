# Numerical precision

All numerical values from result files are automatically rounded to 8 decimal digits.

The numerical precision in the result files is below this threshold (5 decimal digits), so most of the results will not show any difference from 5 to 8 decimal digits (are just trailling zeroes).

I have decided to tighten the precision to avoid float number imprecisions when dealing with the unit conversions.
